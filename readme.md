# Anki Web Editor Toolbar

一款专为 Anki 网页版（ankiuser.net）设计的全能编辑工具栏，解决原生编辑器功能薄弱、操作繁琐的问题，支持 front/back 双输入框无焦点跳转，可自由拖动、自定义显示功能，大幅提升卡片制作效率。

## ✨ 核心亮点

- **双输入框完美适配**：操作 back 输入框时不会跳回 front，两个输入框独立操作，告别焦点劫持困扰

- **功能独立开关**：可自由勾选需要显示的功能按钮，隐藏不常用功能，保持工具栏简洁

- **可自由拖动**：工具栏支持鼠标拖动，可放置在页面任意位置，不遮挡编辑区域

- **全能编辑功能**：涵盖格式设置、颜色、字体大小、列表、表格、代码块等所有卡片制作常用功能

- **轻量无冗余**：仅注入必要代码，不影响 Anki 网页版原生功能，加载速度快

## 🔧 安装步骤（适用于 Edge/Chrome 浏览器）

### 1. 下载源码

克隆本仓库到本地，或直接下载 ZIP 压缩包并解压，确保文件夹内包含以下3个文件：

```plain text
anki-web-editor/
├── manifest.json  # 扩展核心配置文件
├── content.js     # 功能逻辑核心代码
└── style.css      # 工具栏样式文件
```

### 2. 开启浏览器开发者模式

- 打开 Edge/Chrome 浏览器，进入「扩展管理」页面（Edge：设置 → 扩展 → 管理扩展；Chrome：设置 → 扩展程序）

- 开启页面右上角的「开发者模式」开关

### 3. 加载已解压的扩展

- 点击「加载已解压的扩展」按钮，选中刚才解压的 `anki-web-editor` 文件夹

- 加载成功后，浏览器右上角会出现扩展图标（若未显示，可点击扩展图标 → 固定该扩展）

### 4. 生效验证

- 打开 Anki 网页版（[https://ankiuser.net/](https://ankiuser.net/)），登录你的账号

- 进入「添加卡片」页面，按 `Ctrl+Shift+R` 强制刷新页面

- 页面右侧会出现工具栏，即可正常使用所有功能

## 📖 使用方法

### 1. 基础操作

- 点击 front/back 输入框（激活编辑器），选中需要编辑的文字

- 点击工具栏对应的功能按钮，即可完成格式设置

- 未选中文字时，点击需要选中文本的功能按钮，会提示「请先选中文字」

### 2. 工具栏拖动

按住工具栏顶部的「≡ 拖动」区域，鼠标左键按住并拖动，可将工具栏移动到页面任意位置，松开鼠标即可固定。

### 3. 功能开关设置

- 点击工具栏顶部的「功能开关」按钮，弹出功能配置面板

- 面板中每个功能对应一个复选框，勾选=显示该功能按钮，不勾选=隐藏该功能按钮

- 设置完成后，点击「应用」按钮，工具栏会立即更新显示状态

- 再次点击「功能开关」可关闭配置面板

## ⚠️ 注意事项

- 仅支持 Anki 网页版（`https://ankiuser.net/*`），不支持 Anki 桌面客户端

- 首次加载扩展后，需强制刷新 Anki 页面（Ctrl+Shift+R）才能生效

- 若功能失效，可尝试：扩展管理页 → 找到该扩展 → 点击「重新加载」，再刷新 Anki 页面

- 插入表格后，可直接点击单元格编辑内容，表格大小可通过 Anki 原生编辑功能调整

- 代码块默认使用 monospace 字体，灰色背景，可根据需求在 `content.js` 中修改样式
